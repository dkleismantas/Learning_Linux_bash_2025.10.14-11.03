# Linux bash lessons
## Overview
In this module, I learned how to work with Linux through the terminal command line. Although I was already familiar with VirtualBox, I improved my knowledge of it. I also learned how to create a virtual server in Microsoft Azure, how to create a Docker image and container, how to generate an SSH key, and how to host a website on the created server.

# Technologies used
- VirtualBox
- Linux OS
- Terminal
- Microsoft Azure
- Docker

# How to open the assignments
**Warning:** All assignments are provided in Lithuanian.
- Clone the repository.
- Select the folder of the topic you want and open it.
- Inside, you will find a Word document. Open it — it contains the tasks and screenshots showing how they were solved.
